# LabMate AI

LabMate AI is a simple web-based laboratory assistant designed to support students and beginner lab users during experimental workflows.

## Features

- **Experiment Analysis**
  - Select experiment type
  - Add lab notes or experiment issues
  - View possible error sources
  - See things to check
  - Get an organized note
  - Generate a short scientific summary

- **Timer Reminder**
  - Select a common lab step or enter a custom one
  - Set a duration manually
  - Start a countdown for lab steps
  - Receive a reminder when the time is over

- **Download Report**
  - Download experiment analysis results as a `.txt` report

- **Reset Functions**
  - Clear experiment analysis inputs and outputs
  - Reset timer inputs and countdown

## Purpose

The aim of LabMate AI is to make laboratory work more organized, structured, and user-friendly.  
It helps users keep track of time-sensitive steps and review experiment notes in a more systematic way.
